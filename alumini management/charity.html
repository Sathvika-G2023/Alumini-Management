Charity run or walk


A charity run or walk is a fundraising event where participants raise money for a charity by running or walking a set distance. Charity runs and walks can be fun, social, and a great way to combine fitness with helping others.


Charity runs A running event that raises money for a charity
Can be a fun run, 10k race, marathon, or ultra distance challenge
Can be a way to combine fitness goals with helping others

Charity walks
A community fundraiser where participants raise money by walking a set distance
Can be a less-competitive version of a marathon
Can be a way to bring together people who care about the same cause

Tips for participating
Choose a distance that you're comfortable with, but also challenge yourself
Consider what you'll need to bring, like sturdy shoes, layers of clothing, and a backpack
Get friends involved
Do some training
